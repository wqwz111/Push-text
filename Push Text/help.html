说明

  1. 第一次安装插件时, 如果浏览器已经有打开的标签页, 则需要将想使用该插件的标签页刷新才能有效加载该插件。
  2. 打开Chrome后，插件自动与服务器连接，状态显示为Waiting。此时不能进入房间, 请稍作等待, 服务可用时状态变为Ready。此时可以进入房间。
  3. 在Waiting状态时间过久时, 可以点击黄色插头图标强制连接, 此时再次等待即可。
  4. 使用能够连接到google的网络环境可以快速地使服务可用, 此为备用操作。
  5. 网络缓慢时不要重复点击进入房间或发送图片，该操作会导致后台图片过多，并使加载变慢。
  6. 使用完毕请手动离开房间, 谢谢。


使用方法

发送

  1. 点击右上方笑脸, 在面板内输入自定义的一个房间号, 然后点击enter进入房间
  2. 发送时,按快捷键ctrl + alt + z
  3. 发送的内容为鼠标指向的文字内容和当前可见的页面截图.

接收

  • 方法一: 点击右上方笑脸, 点击面板上的new tab打开一个新的标签页, 输入你需要进入的房间号即可.
